# 转换

## transform
### Transform属性应用于元素的2D或3D转换。这个属性允许你将元素旋转，缩放，移动，倾斜
### 语法： transform: none |franform-functions;
|单位|含义|备注
|:---------|:---------|:---------
|none|不进行转换|
|skew|指定对象skew transformation（斜切扭曲）|skew(x轴, y轴)
|rorate|旋转|rorate(角度)
|translate|平移|translate(x轴, y轴)
|scale|缩放|scale(x轴，y轴)
